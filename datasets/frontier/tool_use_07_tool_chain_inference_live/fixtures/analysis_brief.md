你需要根据原始销售记录和调整事件，重建“净销售额 Top 3 品类”报告。

定义：
- 净销售额 = 订单原始金额 + 调整金额总和
- `adjustments.json` 中 `cancelled` 类型的订单：整单作废，不计入净销售额，也不计入有效订单数
- `adjustments.json` 中 `partial_refund` 类型的订单：订单保留，但需要将退款金额从该订单金额中扣除

排序与统计规则：
- 按品类净销售额从高到低排序
- 如果净销售额相同，按品类名字母序升序
- `average_order_amount` = 品类净销售额 / 该品类有效订单数
- `largest_order` 使用“调整后的单笔订单金额”，不是原始金额
